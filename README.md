# Introduction
This is a DE agnostic binary that achieves layout switching behavior similar to GNOME's

# Dependencies
interception-tools

# Installation
Compile the code, then place the binary somewhere in your **$PATH**. Follow this tutorial: **https://ejmastnak.com/tutorials/arch/caps2esc/**. Replacing **caps2esc** with **acyclls -n <num_of_layouts>**.

If you already use a interception-tools plugin, you can pipe the intercept output to multiple binaries as described here: https://wiki.archlinux.org/title/Interception-tools
